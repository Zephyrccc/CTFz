# CTFz-Client

> Front-end program of CTFz platform

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/Zephyrccc/CTFz.git
cd CTFz/Client
npm i
npm run dev
```
